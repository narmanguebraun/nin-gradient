# Gradient

Quick tool to make a 2 colors gradient, and get its code.

- Previews: Background or Text.
- Code: View and copy its CSS or Tailwind CSS code.
- Demo: [nin-gradient.vercel.app](https://nin-gradient.vercel.app)

## Installation

1. Clone the repo:

```bash
   git clone https://github.com/narmanguebraun/nin-gradient.git
```

2. Navigate to the project directory:

```bash
   cd nin-gradient
```

3. Install the dependencies:

```bash
   npm install
```

4. Start the development server:

```bash
   npm run dev
```

5. View the app on the browser:  
   Then open [http://localhost:3000](http://localhost:3000) in your browser.
